# AI-Driven Web Development Course

# 🚀 Добре дошли в AI-Driven Web Development Course!

Това е иновативен, интерактивен курс, управляван изцяло от вашия персонален **AI Ментор** директно във VS Code.

## 🏁 Как да стартирате?

1. Отворете тази папка във **VS Code**.
2. Уверете се, че имате инсталиран **GitHub Copilot** и **GitHub Copilot Chat**.
3. Отворете прозореца на Copilot Chat (`Ctrl+Chit+I` или иконата за чат).
4. Копирайте съдържанието на файла `.github/AI_TEACHER_PROMPT.md` и го изпратете като първо съобщение на AI с текст:
   > *"Здравей, прочети тези инструкции и поеми ролята на мой учител!"*
5. Следвайте стъпките, които AI ще ви даде! 🎓
