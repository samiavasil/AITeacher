
# Advanced HTML Elements

## Таблица с по-рядко използвани HTML тагове

| Категория                | Тагове и пример | Описание |
|--------------------------|-----------------|----------|
| **Мултимедия и описание**| `<figure>`, `<figcaption>` | Групиране на изображение с надпис |
| **Скрито/разгъваемо**   | `<details>`, `<summary>` | Скрит текст, accordion |
| **Маркиране/осветяване** | `<mark>` | Маркира текст (жълто) |
| **Време и измервания**   | `<time>`, `<progress>`, `<meter>` | Дата/час, прогрес бар, измерване |
| **Резултат**             | `<output>` | Показва резултат от изчисление |
| **Код и клавиши**        | `<code>`, `<pre>`, `<kbd>` | Програмен код, преформатиран текст, клавишни комбинации |
| **Цитати и източници**   | `<blockquote>`, `<cite>` | Цитати и източници |
| **Шаблони**              | `<template>` | Дефинира HTML, който не се рендерира веднага |
| **Ruby анотации**        | `<ruby>`, `<rt>`, `<rp>` | Анотации за източноазиатски езици |
| **Вградени обекти**      | `<object>`, `<embed>`, `<param>` | Вграждане на външно съдържание |
| **Списъци с описания**   | `<dl>`, `<dt>`, `<dd>` | Описателни списъци |

## Примери за употреба

### Figure и Figcaption
```html
<figure>
  <img src="pic.jpg" alt="Вкусна храна" />
  <figcaption>Картофи с масло, готвени на пара</figcaption>
</figure>
```

### Details и Summary
```html
<details>
  <summary>Кликни за повече информация</summary>
  <p>Скритото съдържание...</p>
</details>
```

### Mark елемент
За маркиране на текст: `<mark>важен текст</mark>`

### Time елемент
За дати: `<time datetime="2026-05-21">21 май 2026</time>`

### Code, Pre, Kbd
- code: програмни кодове
- pre: преформатиран текст (запазва пространери)
- kbd: клавишни пресекции

### Blockquote, Cite
Цитати: `<blockquote cite="URL"><p>Цитата...</p></blockquote>`
