## Изисквания
1. figure със img и figcaption
2. дета details със summary (поне един)
3. mark на някой текст

5. blockquote със cite атрибут
6. code елемент или pre
7. kbd для показване на клавиши (примерно: натисни <kbd>Ctrl</kbd> + <kbd>C</kbd>)
