# Assignment: Semantic Layout

Построй страна със всички семантични елементи.

## Изисквания
1. header с h1 и навигационни линкове (списък)
2. nav с минимум 3 линка
3. main с два article или section елемента
4. aside с допълнителна информация
5. footer със авторски права

## Съдържание
- Избери тема (блог, магазин, портфолио) и напълни със смислено съдържание
