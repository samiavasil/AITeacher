# Assignment: Images

Створи HTML страница която показва галерия с 3 изображения.

## Изисквания
1. h1 заглавие "Моята Галерия"
2. 3 img елемента с различни изображения от интернет
3. Всяко изображение да има:
   - src: URL към изображение
   - alt: смислен описателен текст
   - width: фиксирана ширина, напр. 300px

## Бонус
Добави p елементи между изображенията с описание на всяко.
