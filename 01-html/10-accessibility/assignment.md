# Assignment: Accessibility

Построй достъпна страница с правилни практики.

## Изисквания
1. Правилна heading структура (h1, h2, h3)
2. Всички img с alt текст
3. Всички input със label
4. Семантични елементи (header, nav, main, footer)
5. Добър контраст на текст
6. Всички функционалности да са keyboard-достъпни

## Съвет
Тестирай с keyboard-only навигация (Tab, Enter, Shift+Tab).
