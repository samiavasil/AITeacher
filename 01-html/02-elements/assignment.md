# ASSIGNMENT: HTML Elements

Направи страница за своята гимназия, като използваш правилните HTML елементи за правилното съдържание.

## Изисквания
1. `h1` за името на гимназията
2. `p` за кратко описание от 1-2 изречения
3. `h2` със заглавие `Специалности`
4. `ul` с поне 3 `li` елемента за специалности
5. `a` линк към официален сайт на гимназията

## Съвет
Избирай елемента според значението на съдържанието, не според това как изглежда.
