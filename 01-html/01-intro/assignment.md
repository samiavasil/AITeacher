# ASSIGNMENT: HTML Intro

Създай първата си HTML страница и покажи, че разбираш базовата структура на документа.

## Изисквания
1. Отвори `index.html` в тази папка.
2. Добави валидна HTML структура с:
	- `<!doctype html>`
	- `<html>`
	- `<head>`
	- `<body>`
3. В `<head>` добави:
	- `<meta charset="UTF-8" />`
	- `<title>` с кратко заглавие
4. В `<body>` добави:
	- едно `h1` заглавие
	- един `p` абзац с кратко представяне

## Съвет
Фокусирай се върху правилната структура, а не върху красив дизайн.
